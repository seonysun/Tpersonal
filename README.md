  * 자바스크립트 오류 유형 *
    - EvalError : eval() 함수 관련 오류
    - InternalError : 자바스크립트 엔진 내부 오류
    - RangeError : 숫자/매개변수 범위가 유효하지 않음
    - ReferenceError : 참조 오류
    - SyntaxError : 구문 오류
    - TypeError : 변수/매개변수의 자료형이 유효하지 않음
    - URIError : encode/decode URI 의 매개변수 부적절
